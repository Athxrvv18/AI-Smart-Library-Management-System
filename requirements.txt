## 📦 Requirements

This project uses the following Python libraries:

```txt
opencv-python
pyzbar
mysql-connector-python
RPi.GPIO
pillow
RPLCD
smbus2
```

### 🔍 Library Purpose

* **opencv-python** → Used for capturing images from the camera
* **pyzbar** → Used for decoding barcodes and QR codes
* **mysql-connector-python** → Connects Python with MySQL database
* **RPi.GPIO** → Controls Raspberry Pi GPIO pins (IR sensors)
* **pillow** → Supports image processing operations
* **RPLCD** → Controls the 16x2 LCD display
* **smbus2** → Enables I2C communication for LCD

### ⚙️ Installation

Install all dependencies using:

```bash
pip install -r requirements.txt
```
